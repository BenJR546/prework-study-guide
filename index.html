HTML

  • HTML is a framework that a browser will read and interpret as a webpage. To further style and give the page interractibility, you'll link it with CSS and JavaScript code.
  • HTML works with closed 'elements', which are tags that define how the information should be displayed. For example, the "HTML" title shown above this section will appear as "<h2>HTML</h2>"
  • The head element contains information about the webpage.
  • The body element represents the visable content shown to the user.
