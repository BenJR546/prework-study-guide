HTML

  • HTML is a framework that a browser will read and interpret as a webpage. To further style and give the page interractibility, you'll link it with CSS and JavaScript code.
  • HTML works with closed 'elements', which are tags that define how the information should be displayed. For example, the "HTML" title shown above this section will appear as "<h2>HTML</h2>"
  • The head element contains information about the webpage.
  • The body element represents the visable content shown to the user.

CSS

  • CSS is a stylesheet language used for styling web pages. It defines how HTML elements should be displayed.
  • Basic Concepts:
  • Selectors: CSS selectors target HTML elements (e.g. Tags, classes, IDs)
  • Properties: CSS properties define styling rules (e.g. colour, font size, margin)
  • Values: CSS values determine the specific style (e.g. green, 20px, left)
  • Important CSS Commands:
  • selector {property: value;}: Basic CSS rule structure
  • color: Sets the text colour
  • font-size: Defines the font size
  • margin and padding: Control spacing around elements
  • background-color: Sets the background colour
  • border: Adds borders around elements
  • display: Determines how an element is displayed (e.g. block, inline, flex)

Git

  • Git is a language used to interact between files saved locally on your device and online on "GitHub"
  • Some basic concepts of Git are:
  • Repositories (or Repos)- A directory where Git tracks changes
  • Commit-A snapshot of changes made to files
  • Branch-A seperate line of development
  • Merge-Combining changes from one branch into another
  • Some Important Git Commands are:
  • git init: Initializes a new Git Repository
  • git clone <repository>: Downloads an existing repositry
  • git add <file>: Stages changes for commit
  • git commit -m "message": Saves changes with a descriptive message
  • git status: Shows the status of files in the repository
  • git log: Displays commit history
  • git branch: Lists branches
  • git checkout <branch>: Switches to a different branch
  • git merge <branch>: Merges changes from one branch to another
  • git pull: Fetches changes from a remote repository
  • git push: Uploads local changes to a remote repository

JavaScript

  • JavaScript is a high-level, interpreted programming language primarily used for adding interactivity to web pages. It runs in web browsers and allows for dynamic content manipulation
  • Basic Concepts:
  • Variables: Declare and store data (e.g. numbers, strings, objects)
  • Funtions: Blocks of reusable code
  • Conditional Statements: Execute code based on conditions (e.g. if, else, switch)
  • Loops: Repeat code (e.g. for, while)
  • Important JavaScript Commands:
  • console.log("message"): Outputs messages to the browser console
  • document.getElementByID("elementId): Accesses a HTML element by its ID
  • let and const: Declare variables (with block scope)
  • funtion myFunction() {...}: Defines a function
  • if (condition) {...} else {...}: Conditional Statement
  • for (let i = 0; < n; i++){...}: Loop Structure
  • addEventListener("event", callback): Listens for events (e.g. clicks, key presses)
  • JSON.parse() and JSON.stringify: Manipulate JSON data
  • DOM Manipulation
  • DOM (Document Object Model) represents the structure of a HTML document. Use JavaScript to manipulate DOM elements (e.g. change content, add/remove elements)
